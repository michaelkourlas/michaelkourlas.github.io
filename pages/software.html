Software

I have written the following software applications and libraries:

  • js2xmlparser, a Node.js module that parses JavaScript objects to XML;
  • VoIP.ms SMS, an Android application that sends and receives SMS messages using a VoIP.ms account;
  • cloudflare-dynamic-dns, a Node.js module that updates a Cloudflare DNS address record with an IP address; and
  • VoIP.ms SMS Server, a companion server that sends push notifications to VoIP.ms SMS users using Google Cloud Messaging in response to SMS callbacks from VoIP.ms;
  • xmlcreate, a Node.js module that can be used to easily build XML using a simple API.
